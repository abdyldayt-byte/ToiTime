# ТойTime 01 — Invitation Landing Page

Новый формат праздника в Кыргызстане. Веб-приглашение для события "Незнакомые люди".

## Особенности

- Одностраничный сайт без зависимостей
- Чёрно-золотой дизайн в стиле премиум-события
- Интерактивное открытие приглашения
- Плавные анимации и reveal-эффекты при прокрутке
- WhatsApp интеграция для бронирования билетов
- Полностью адаптивный (мобильная и desktop версии)
- Всё в одном HTML файле (Tailwind CSS через CDN)

## Структура

```
ToiTime/
├── index.html          # Основной сайт
├── vercel.json         # Конфиг для Vercel
├── .gitignore
└── .claude/
    └── launch.json     # Конфиг для preview
```

## Локальный запуск

Требует любого HTTP-сервера:

```bash
npx serve -l 5500 .
# или
python -m http.server 8000
```

Откройте http://localhost:5500 в браузере.

## Деплой на Vercel

### Вариант 1: Через веб-интерфейс Vercel

1. Перейди на [vercel.com](https://vercel.com)
2. Нажми "Add New" → "Project"
3. Выбери репозиторий `ToiTime`
4. Framework: выбери "Other" (статический сайт)
5. Deploy

Vercel автоматически пересроит сайт при каждом push в `main`.

### Вариант 2: Через Vercel CLI

```bash
npm i -g vercel
vercel
```

Следуй инструкциям CLI — Vercel создаст проект и развернёт автоматически.

## Содержание сайта

- **Открывающий экран**: интерактивное приглашение с кнопкой "Открыть"
- **Hero секция**: название события, дата/время, место, основная информация
- **О событии**: концепция ТойTime, что входит, целевая аудитория
- **Программа**: что ждёт гостей, программа вечера
- **Преимущества**: что оставили, что убрали
- **Билеты**: стоимость, варианты предоплаты, ограничение мест
- **Бронирование**: кнопки для WhatsApp и телефонного звонка
- **Плавающая кнопка WhatsApp**: доступна везде на сайте
